Beim Start des Refactorer kann der Treshhold sowie die Anzahl der zu bearbeitenden Bilder angegeben werden: 
path/to/Refactorer.jar java -jar Reafactorer.jar 23 all
-> Treshhold wird auf 23 gesetzt
-> alle Bilder des Ordners werden bearbeitet

Die Refactorer.jar muss sich in einem Ordner befinden, der Bilder mit dem Namen xIMGy.JPG

x-> Wertindex der Münze
y-> Nummer der Münze
